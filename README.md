Welcome to my personal portfolio website. This project is a place where I showcase my skills, projects, and journey as a frontend developer. It reflects my passion for building modern, responsive, and user-friendly web experiences.

## About the Project

This portfolio was built to create a professional online presence and showcase the projects I've worked on. My goal was to design a clean, modern, and responsive website that provides visitors with an overview of my skills, experience, and work.

## Features

- Dark and Light Mode
- Fully Responsive Design
- Single Page Application (SPA)
- Project Showcase Section
- Skills Section
- Contact Form
- Smooth Navigation with React Router
- Basic UI Animations
- Modern and Clean User Interface

## Built With

- React JS
- Tailwind CSS
- React Router

## What I Learned

While building this project, I strengthened my understanding of:

- Responsive web design
- Routing with React Router
- State management
- UI/UX design principles
- Creating reusable and maintainable code

## Getting Started

To run this project locally:

```bash
git clone https://github.com/ataullaharabzai/Portfolio-website.git

cd portfolio

npm install

npm run dev
```
